# Student Performance Prediction

This project predicts students' final grades using machine learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
Linear Regression

## Features Used
- studytime
- failures
- absences
- G1 (first period grade)
- G2 (second period grade)

## Model Evaluation
Mean Absolute Error ≈ 1.34

## Result
The model predicts student grades with reasonable accuracy.

## Dataset
Student Performance Dataset from UCI Machine Learning Repository.
